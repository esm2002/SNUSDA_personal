5. Pytorch 2.2.1 설치 (공식문서 들어가서 pip 명령어 복붙) -> pytorch 2.2.1에서 networkX 라이브러리가 파이썬 3.9 이상부터 지원한다는 에러가 뜸. 하지만 networkX는 파이썬 3.8이 지원하는 라이브러리로 별도로 설치해도 상관없다고 해서(공식 큐앤에이 참고 - https://discuss.pytorch.org/t/installing-pytorch-under-python-3-8-question-about-networkx-version/196740), 그것만 별개로 설치할 예정. pip install networkx==3.1 명령어로 설치 완료함. 

6. 텐서플로우 정상 설치 확인 python3 -c "import tensorflow as tf; print(tf.__version__)" 실행: 2.13.1 설치 완료. 그러나 Warning: Could not find TensorRT이 뜸. 만약 소스코드 돌려볼 때 텐서플로우 문제 생기면 이 문제부터 해결해보기.

7. 파이토치 정상 설치 확인 pip show torch 실행: 이게 어떤 이유로 파이토치 패키지 정보를 확인할 수 없다고 뜨는데, 파이토치 설치 자체는 networkx 제외하고 잘 되었고, 이후에 networkx도 버전 낮춰서 잘 설치했다고 뜸. 일단 소스코드 돌려보도록 하자.
